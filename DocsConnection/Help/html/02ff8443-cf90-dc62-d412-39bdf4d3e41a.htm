DatabaseConnection Methods

The DatabaseConnection type exposes the following members.

Methods
 NameDescription
CloseSqlConnection Closes the SQL connection if it is open.
ExecuteSqlNonQuery Executes a SQL non-query command (e.g., INSERT, UPDATE, DELETE).
GenerateQuery Generates a SQL query command using a table name and a dictionary of field values.
OpenSqlConnection Opens the SQL connection if it is not already open.
RetrieveAllDataFromTable Retrieves all data from a specified table.
RetrieveDataUsingQuery(SqlCommand) Retrieves data using a specified SQL command.
RetrieveDataUsingQuery(String) Retrieves data using a specified SQL query.
RetrieveDataUsingQuery(String, String) Retrieves data using a specified SQL query and a table name to structure the DataSet.
UpdateData Updates data in the database using a specified SQL query and a DataSet.
